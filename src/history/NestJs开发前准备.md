# NestJs开发前准备

> 确保操作系统上安装了 Node.js（ >= 8.9.0）

## 全局安装依赖

```shell

yarn global add @nestjs/cli
yarn global add @nestjs/schematics

```

## 使用脚手架创建基础项目

```shell

nest new template

```
