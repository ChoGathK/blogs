# JavaScript 基础篇介绍

> 对于 Node.js / Web 开发者来说，能够理解并熟练 JavaScript 的基础知识和特性是非常必要的。因为实际工作中，具体的需求使我们不得不直面对各类场景的挑战，去应对并充分发挥 JavaScript 的特性和优点，规避语言特性导致的缺陷。而随着项目或者业务的复杂度提升，又对开发者提出了更高的挑战。

**在基础篇中，我们将一起学习 Javascript 中的核心基础知识。**

## 历史

在上个世纪的1995年，当时的网景公司凭借其 `Navigator` 浏览器，成为 Web 时代早期最著名的第一代互联网公司。后来，网景公司希望能在静态HTML页面上添加一些动态效果，于是 Brendan Eich 老哥在两周之内设计出了 JavaScript 语言。

**为什么起名叫JavaScript？**

原因是当时Java语言非常红火，所以网景公司希望借 Java 的名气来推广，但事实上 JavaScript 除了语法上有点像 Java，其他部分基本无关。

## 组成部分

**语言核心**

`ECMAScript`：是一种语言标准，而 JavaScript 是网景公司对 ECMAScript 标准的一种实现。

**运行时环境**

- `Web浏览器`：BOM、DOM
- `Node.JS`：网络、文件系统、IO

## 核心知识

- `基础`：语言特性、语法、数据类型、内存分配、隐式转换
- `函数和作用域`：函数、作用域、闭包函数、高阶函数
- `面向对象`：原型、原型链、继承、类、this 指针
